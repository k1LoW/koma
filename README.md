# [PoC]Koma

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'bun', github: "k1LoW/koma"
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install specific_install
    $ gem specific_install https://github.com/k1LoW/koma.git 

## Usage

    $ koma ssh example.com
